前言

工作原因需要远程操作 Windows 10，但 AppStore 上的 Microsoft Remote Desktop 却提示地区不支持下载，更换美区 ID 问题依旧。目前找到两个方法可以下载该应用的安装包，记录一下。

第三方网站

Microsoft Mac Downloads 网站上有微软 Mac 版的各类软件，甚至连历史版本都有，非常方便。

这里放出最新版（February 7, 2022） Microsoft Remote Desktop 的下载直链，可直接点击下载。

官方最新测试版

在微软官方文档里搜索「Microsoft Remote Desktop for Mac」，点击进去就是官方预览版的下载页面，选择最新版本下载即可。
